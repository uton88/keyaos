# Quick Start

Get up and running with Keyaos in under two minutes.

## 1. Get Your API Key

Sign in to your dashboard and generate a key from the [API Keys page](/dashboard/api-keys). Your full key is shown only once — copy and store it securely.

## 2. Choose Your SDK

Keyaos supports both OpenAI and Anthropic SDKs. Pick the one that fits your workflow.

### OpenAI SDK

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-keyaos-xxxxxxxx",
    base_url="https://keyaos.com/v1",
)

response = client.chat.completions.create(
    model="openai/gpt-4o-mini",
    messages=[{"role": "user", "content": "What is Keyaos?"}],
)
print(response.choices[0].message.content)
```

### Anthropic SDK

```python
import anthropic

client = anthropic.Anthropic(
    api_key="sk-keyaos-xxxxxxxx",
    base_url="https://keyaos.com",
)

message = client.messages.create(
    model="anthropic/claude-sonnet-4",
    max_tokens=1024,
    messages=[{"role": "user", "content": "What is Keyaos?"}],
)
print(message.content[0].text)
```

### Embeddings

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-keyaos-xxxxxxxx",
    base_url="https://keyaos.com/v1",
)

response = client.embeddings.create(
    model="qwen/qwen3-embedding-4b",
    input="The quick brown fox jumps over the lazy dog",
)
print(f"Dimensions: {len(response.data[0].embedding)}")
```

### Reasoning Models

Control how much a model "thinks" before answering using `reasoning_effort`:

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-keyaos-xxxxxxxx",
    base_url="https://keyaos.com/v1",
)

response = client.chat.completions.create(
    model="openai/o4-mini",
    messages=[{"role": "user", "content": "Solve: x^2 + 3x - 10 = 0"}],
    extra_body={"reasoning_effort": "high"},
)
print(response.choices[0].message.content)
```

See [Models & Routing](/docs/models-routing#reasoning-effort) for all supported formats and providers.

### Image Generation

Generate images via the same chat completions endpoint:

```python
response = client.chat.completions.create(
    model="openai/gpt-5-image-mini",
    messages=[{"role": "user", "content": "A watercolor painting of a sunset over the ocean"}],
    extra_body={"modalities": ["image", "text"]},
)
```

See [Multimodal](/docs/multimodal) for image inputs, PDFs, audio, and video.
